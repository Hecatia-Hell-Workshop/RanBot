### 快速部署

---
1.下载Bot本体

```shell
git clone https://github.com/Hecatia-Hell-Workshop/RanBot.git # 下载Bot文件
```
2.安装依赖(**强烈建议**使用你喜欢的 Python 环境管理工具创建新的虚拟环境)
```shell
cd RanBot # 进入文件夹

pip install pip --upgrade # 更新pip

pip install -r requirements.txt # 安装python依赖
```
3. 启动RanBot！
```shell
python ./boot.py # 启动bot！
```
